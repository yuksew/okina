# P1ゲート判定の診断: なぜ全戦略が合格ライン未達だったか

- 作成日: 2026-07-10
- 根拠データ: `pnpm run explain`（年次リターン・DD局面・要因分解）および [07-backtest-report.md](07-backtest-report.md)
- 共通ウィンドウ: 2006-01-27 〜 2026-07-09、コスト込み・税抜き

## 結論の要約

未達の原因は戦略ごとに異なり、**「実装バグ」や「偶然」ではなく構造的な理由**がある。同時に、評価条件の側にも2つの検証可能な保守バイアス（コスト仮定・評価ウィンドウの性質）があり、これはパラメータいじりではなく事実確認で解消できる。

## 1. S1（SPY70/AGG30）: DD 39.6% は「静的な株式βの代金」

- 最悪DDは 2007-10→2009-03 の**リーマン危機そのもの**（SPY単体は-55.2%、その70%を常時保有すれば約-39%になるのは算術）。
- 次点も 2020コロナ(-24.2%)、2022(-21.8%) と、株式暴落がそのまま貫通している。
- つまりS1は「壊れている」のではなく、**DD20%制約と静的70%株式は原理的に両立しない**。年次リバランスはDDをほぼ軽減しない（リバランスは配分維持の道具であり、下落防御の道具ではない）。

## 2. S2（GTAA5+10ヶ月SMA）: DDは設計通り抑えたが、リターンを3つの要因で失った

要因分解（SPY単体B&H / SPY+フィルタ / GTAA5等ウェイト・フィルタ無 を別々に実測）:

| 要因 | 実測根拠 | 影響 |
|---|---|---|
| (a) ユニバース分散の逆風 | GTAA5等ウェイト(フィルタ無)は2013年 +2.5% vs SPY +32.3%。2013〜2024は米国大型株の独走相場で、EFA/VNQ/GLD/IEFへの分散はこの期間**一貫してリターンを削った** | 大（最大要因） |
| (b) 速い暴落でのダマシ（whipsaw） | SPY+フィルタは2008年 0.0% vs SPY -36.8% と**リーマンでは完璧に機能**した一方、2010(-3.5% vs +15.1%)、2019(+8.1% vs +31.2%)、2022〜23(-23.4%/+6.9% vs -18.2%/+26.2%)では「下がってから売り、上がってから買う」を繰り返した。**2022年はB&Hより深く負けている**（ジリ下げ＋ベアラリーの繰り返しはSMAフィルタの最悪パターン） | 中 |
| (c) コスト | 回転率201%/年 × 往復1.2% ≒ **約1.2%/年のドラグ**（コスト累計447千円/初期1,000千円） | 小〜中 |

- 重要な非対称性: トレンドフィルタは「保険」であり、保険料（ダマシ）は毎年払うが、支払い事由（2008年型のゆっくり深い暴落）は**この20年で1回しか起きていない**。2018/2020/2022の暴落はいずれも速すぎ・浅すぎ・回復が急で、保険が効かないタイプだった。
- それでも最大DD 14.8%・パラメータ8/10/12ヶ月で結論不変というDD制御の頑健さは確認できた。**S2は「リターンを犠牲にDDを買う」という設計意図どおりに動いている**。

## 3. S3（GEM）: この時代に対して構造的に不利な2点で明確に不合格

1. **100%集中 × 12ヶ月モメンタムは速い暴落に間に合わない**。最悪DD 35.7%（2018-01→2020-03）の中身: 2018Q4急落で月末判定が遅れて退避→2019年の急回復に乗り遅れ（+9.4% vs SPY +31.2%）→2020年2月末時点では12ヶ月モメンタムがまだ正で株式保有のままコロナ暴落を直撃→3月末に底値圏で退避＝損失確定→4月からのV字回復を取り逃す。**「遅れて逃げ、遅れて戻る」を全資産100%で行う**ため、1回の判定ミスが致命傷になる。
2. **相対モメンタム（SPY vs EFA）がこの期間は無価値だった**。2008年以降EFAがSPYを持続的に上回った局面がほぼなく、切替は回転率（409%/年、コスト累計1,036千円）だけを生んだ。GEMの好成績は主に1970〜2000年代の国際分散が効いた時代のデータに依拠しており、文献で指摘される「発表後の劣化」がそのまま再現された。
- 感応度（6/9/12ヶ月）でも全滅しており、パラメータの問題ではない。**候補から除外を推奨**。

## 4. S4（Top3ローテーション）: 集中を分散に置き換えたS3の改良版として惜しい位置

- 3/6/12ヶ月ブレンドで反応が速く、Top3分散で1判定ミスの被害が1/3になるため、DDは20〜22%帯に収まる（S3の35%超とは質的に違う）。Top4なら19.6%で合格圏。
- ただし回転率436%/年は4戦略中最悪で、**コスト仮定と税に最も敏感**。税込みでの再評価なしに採用判断はできない。

## 5. 評価条件側の保守バイアス（2点、いずれも「事実確認」で解消可能）

1. **コスト仮定が重い可能性**: 片道0.595%（手数料0.495%+スリッページ0.1%）を全取引に課している。国内ネット証券は米国ETFの買付手数料無料枠が拡大しており、実勢はこれより大幅に低い可能性がある。回転率200〜436%の戦略には**年1.2〜2.6%ptの逆風**として効いており、S2/S4のシャープを体感で0.1前後押し下げている。→ 対応: 実際に使う証券会社の手数料体系を裏取りして実勢値で再計算する（基準緩和ではなく入力データの補正）。
2. **評価ウィンドウがベンチマークに有利な20年だった**: 2009〜2024は米国株の歴史的強気相場で、S1（=米国株β）のシャープ0.72はこの追い風込み。かつ暴落は速い型ばかりでトレンド系の保険が効かなかった。**「シャープ≥S1」は実質的に「史上最良期の米国株に勝て」という要求**になっている。→ 対応: 別レジーム（例: 2000〜2012を含む指数系列での補助検証、またはブートストラップ）を見るか、基準の妥当性自体をユーザーが再判断する。

## 6. 次の一手の選択肢（ユーザー判断待ち）

- A. **実勢コストの裏取り＋税込みモデル実装＋S2/S4ブレンド追加 → 再判定**（モデル入力の補正であり基準緩和ではない。推奨）
- B. S2を採用してP2へ（DD制御を最優先する判断。CAGR約5%と水面下3年を受容）
- C. 不合格として戦略再設計（S1ベースで株式比率を下げる/ボラターゲット等）
- D. 基準の見直し（「シャープ≥S1」→「DD20%以内の選択肢中最良」等。緩和にあたるため要自問）
