## 1. 結論: 片道コスト率の実勢（外貨決済+リアルタイム為替前提、1ドル=150円）

| ケース | 内訳（片道） | 目安 |
|---|---|---|
| 大手3社（SBI/楽天/マネックス）特定口座 | 手数料0.495%（上限22ドルは約定67万円未満では効かない）+ 為替0銭 + 半スプレッド0.01% | **約0.50%** |
| 同・円貨決済 | + 25銭（≒0.17%） | 約0.67% |
| 買付無料ETF活用（楽天が最多: SPY/QQQ/AGG該当） | 買い0.01% / 売り0.51%。9本ユニバース加重平均 | 約0.42% |
| **moomoo証券 特定口座** | 手数料0.132%（上限22ドル）+ 為替0銭 + 0.01% | **約0.14%** |
| Webull 特定口座 | 現行0.22% → **2026年7月下旬に無料化と報道**（公式未反映）。為替15銭 | 約0.11%（無料化後・報道ベース） |
| NISA成長投資枠（SBI/楽天、売買無料） | スプレッドのみ | **約0.01〜0.05%** |

- SEC Fee（売却時0.00206%）は無視できる規模。
- **含意**: 回転率200〜440%/年の戦略（S2/S4）を大手3社の特定口座で回すと年1.0〜2.2%のコスト、moomooなら年0.3〜0.6%に圧縮。**サテライトの実行先はmoomoo/Webullが合理的で、これは将来のAPI自動化候補（02-research §1）と同一**。コアNISA（SBI/楽天）は低回転前提でコストほぼゼロ。
- **追記（2026-07-12）**: moomooは金融庁の一部業務停止命令により新規口座開設を2026-09-18まで受付停止。サテライト執行先は**ウィブル証券へ変更**（[10-p1-gate-decision.md](10-p1-gate-decision.md) 改訂履歴参照）。

## 2. NISA・無料プログラムの詳細

- NISA口座: SBI・楽天は米国株/ETF売買完全無料、マネックスは全額キャッシュバック方式（一次確認済み）。
- 買付無料ETF（買いのみ無料、売りは0.495%）: SBI「ETFセレクション」10銘柄（QQQ/AGG該当、GLD代替のGLDMあり）、楽天15銘柄（SPY/QQQ/AGG該当）、マネックス「買い放題」22銘柄（リスト取得不可・想定9本の包含は未確認）。

## 3. バックテストへの反映値

| モデル | commissionRate | slippageRate | 用途 |
|---|---|---|---|
| NISA_COSTS | 0% | 0.05%（保守） | S1コア（NISA） |
| MAJOR3_COSTS | 0.495% | 0.02% | シナリオ「大手3社・特定」 |
| MOOMOO_COSTS | 0.132% | 0.02% | シナリオ「moomoo・特定」 |

- スリッページは高流動性ETFの半スプレッド実勢（0.5〜2bp）に対し保守側の2bp。
- 為替コストは外貨決済+リアルタイム為替（0銭）前提で0。円貨決済しか使わない場合は片道+0.17%が必要（採用しない前提）。

## 4. 裏取り不可・実装/口座開設前に要再確認

1. マネックス「買い放題」現行22銘柄の完全リスト（公式ページがbot遮断）
2. moomooの円貨決済為替手数料（0銭説と25銭表記が混在。外貨決済なら0銭で確実）
3. Webull手数料無料化の公式実施日・適用条件（2026-07-09時点で報道のみ）
4. SBI「ETFセレクション」リストの最新版（公開ページの時点注記が2023年）
