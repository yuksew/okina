# ETFユニバース定義

- 作成日: 2026-07-10
- ステータス: 初版（S1/S2用は確定、S3/S4用は候補）
- 実装上の定義: `apps/backtest/src/universe.ts`（本書と同期させること）

## 選定基準

1. 15年以上の価格履歴（2008年金融危機を含められること）
2. 十分な流動性と経費率の低さ（同カテゴリで最も標準的なもの）
3. 国内ネット証券で購入可能な米国上場ETFであること（半自動運用の前提）

## マスター一覧

| ティッカー | 資産クラス | 設定年 | 主な用途 |
|---|---|---|---|
| SPY | 米国株式（S&P500） | 1993 | S1/S2/S3/S4 |
| QQQ | 米国株式（NASDAQ100） | 1999 | S4候補 |
| EFA | 先進国株式（除く米国） | 2001 | S2/S3/S4候補 |
| EEM | 新興国株式 | 2003 | S4候補 |
| AGG | 米国総合債券 | 2003 | S1/S3 |
| IEF | 米国債7-10年 | 2002 | S2/S4 |
| TLT | 米国債20年+ | 2002 | S4候補 |
| SHY | 米国債1-3年（現金代替） | 2002 | S3絶対モメンタム基準 |
| VNQ | 米国リート | 2004 | S2 |
| GLD | 金 | 2004 | S2 |

- 共通履歴の最古起点は GLD（2004-11）に律速され、**約2005年〜 ≒ 21年分**を確保（15年要件を充足、2008年を含む）。
- VEA/VWO/BND 等バンガード系は経費率で優るが設定が2007年以降で履歴が短いため、検証は上記で行う。**実運用時に同カテゴリの低コスト代替へ置換してよい**（シグナルは検証銘柄で計算し、執行だけ代替銘柄にする運用も可）。

## データソース

- **主**: Tiingo（adjClose、公式API）。`TIINGO_TOKEN` 取得後に `pnpm run fetch --universe all --source tiingo`
- **クロスチェック**: Stooq を予定していたが、**2026-07-10 実測でCSVエンドポイントがJavaScriptボット検証を返し、サーバーサイド取得は不可**（UA偽装でも同じ）。ブラウザからの手動ダウンロードは可能なため、品質クロスチェックは手動DLしたCSVで行う（`fetch-stooq.ts` はHTML応答を明確なエラーにする実装のまま残置）
- 品質チェック: 両ソースの adjClose 系列の日次リターン相関・乖離を確認してから本検証に入る（P1手順3）
